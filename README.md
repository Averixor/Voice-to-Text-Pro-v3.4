# Голос в Текст Pro 3.3

Продвинутое расширение для преобразования голоса в текст с AI-пунктуацией и проверкой орфографии в формате боковой панели.

## Возможности

### 🎤 Распознавание речи

- Поддержка 10+ языков (русский, английский, украинский, испанский, французский, немецкий, итальянский, японский, корейский, китайский)
- Высокая точность распознавания
- Непрерывная запись с автоматическим перезапуском
- Поддержка промежуточных результатов

### ✨ Умная пунктуация

- 4 уровня пунктуации (Выкл, Низкий, Средний, Высокий)
- Автоматическая капитализация предложений
- Интеллектуальное добавление знаков препинания
- Форматирование текста в реальном времени

### 🔤 Проверка орфографии

- Встроенная проверка орфографии
- Поддержка языковых словарей
- Мгновенное исправление опечаток

### 🎛️ Оптимизированный интерфейс

- Боковая панель для удобного доступа
- Темная тема с анимированным фоном
- Панель инструментов форматирования
- Статистика в реальном времени

### ⌨️ Горячие клавиши

- `Ctrl+Shift+1` - Начать запись
- `Ctrl+Shift+2` - Остановить запись
- `Ctrl+Shift+I` - Вставить текст в активное поле
- `Ctrl+Shift+9` - Открыть/закрыть боковую панель
- `Ctrl+C` (в поле редактирования) - Копировать текст
- `Ctrl+Del` - Очистить текст

## Установка

1. Скачайте все файлы расширения из репозитория
2. Откройте Chrome и перейдите в `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами расширения
6. Расширение появится на панели инструментов

## Использование

1. Нажмите на иконку расширения в панели инструментов Chrome
2. Боковая панель откроется справа
3. Выберите язык распознавания из выпадающего списка
4. Выберите уровень AI-пунктуации
5. Нажмите "Начать запись" или используйте горячую клавишу `Ctrl+Shift+1`
6. Говорите четко и разборчиво
7. Нажмите "Стоп" для завершения записи
8. Используйте кнопки "Вставить", "Копировать", "Очистить" для управления текстом
9. Текст автоматически сохраняется между сессиями

## Особенности боковой панели

- Всегда доступна во время работы с любой вкладкой
- Не мешает основному контенту страницы
- Автоматическое сохранение текста
- Быстрый доступ через горячие клавиши
- Контекстное меню для вставки текста в любом поле ввода

## Решение проблем

### Нет доступа к микрофону

1. Нажмите на значок замка 🔒 в адресной строке
2. Выберите "Настройки сайта"
3. Разрешите доступ к микрофону
4. Обновите страницу

### Не работает распознавание

- Проверьте подключение к интернету (Web Speech API требует соединения)
- Убедитесь, что микрофон работает и доступен
- Перезапустите расширение или переустановите его
- Проверьте настройки конфиденциальности в Chrome

### Текст не вставляется

- Убедитесь, что активное поле ввода поддерживает вставку текста
- Некоторые сайты (Google Docs) могут требовать дополнительных разрешений
- Используйте кнопку "Вставить" на панели инструментов

## Технологии

- Web Speech API - распознавание речи
- Chrome Extensions Manifest V3 - современная платформа расширений
- Chrome Side Panel API - боковая панель
- CSS Grid & Flexbox - современная верстка
- Modern JavaScript (ES6+) - чистый и эффективный код
- Chrome Storage API - сохранение настроек и текста

## Системные требования

- Google Chrome версии 114 или выше
- Микрофон (встроенный или внешний)
- Активное интернет-соединение (для Web Speech API)
- 50 MB свободного дискового пространства

## Обновление

Для обновления расширения:

1. Скачайте новую версию файлов
2. В `chrome://extensions/` нажмите "Обновить" на карточке расширения
3. Или удалите расширение и установите заново
